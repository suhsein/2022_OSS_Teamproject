## * Feature/4 구현점
    학과의 이름을 영문으로 쓰고, 그 학과 사무실의 위치를 출력한다.
    오탈자가 있을 경우를 위해 levenshtein 알고리즘을 응용해서 가장 비슷한 문자열로 변환해서 확인한다.
    확인을 y,n으로 확인을 받아 최종적으로 출력한다.
